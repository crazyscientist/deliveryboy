Installation
============

For now this project is only hosted on GitHub.

Requirements
------------

- `dill <https://pypi.org/project/dill>`_

Installation
------------
via GitHub ...
^^^^^^^^^^^^^^
clone it, run it

.. code-block:: python

    git clone https://github.com/crazyscientist/deliveryboy.git

... or pip
^^^^^^^^^^

.. code-block:: bash

    pip install deliveryboy

Configuration
-------------
None so far
